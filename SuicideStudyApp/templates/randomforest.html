Suicide Rates Study

Random Forest Regression

October, 2019 by Caroline

A machine learning model that was used in this study was a random forest regression model from Scikit Learn to predict suicide rates based on population, gdp for year, gdp per capita. A random forest regression is of the ensemble learning type, which means that it uses various algorithms to arrive to a final prediction. A random forest model creates smaller decision trees from the data to make a final prediction.

After outliers were removed from the data, the data was seperated into training data and testing data. The data was split so that the performance of the model can be tested before the application went into production. The training data was used to trained the data and the testing data was used to test the data.

A Standard Scaler was used to scale the training and testing data so that data of different units matched one another.

The training data returned a score of 0.90 and the testing score returned a score of 0.54. The training score of the random forest regression was too high, which indicated that the model was overfitting. The test score was average because of the model was overfitting Population, GDP for year, and GDP per Capita may not the best predictors of suicides number
