Suicide Rates Study

Support Vector Machine Regressor

October, 2019 by Caroline

A machine learning model that was used in this study was support vector machine regression to predict suicide rates with population, GDP for year, and GDP per Capita. A support vector machine regression uses smaller datasets to categorize data accross a lane.

After outliers were removed from the data, the data was seperated into a training data and testing data. The data was split so that the performance of the model can be tested before the application goes into production. The training data was used to trained the data and the testing data was used to test the data.

A Standard Scaler was used to scale the training and testing data so that data of different units matched one another.

The training data returned a score of 0.44 and the testing score returned a score of 0.42. Both the training and testing scores were average. Population, GDP for year, and GDP per Capita may not the best predictors of suicides number
